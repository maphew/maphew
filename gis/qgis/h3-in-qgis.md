# Qgis & H3
Using Uber's H3 hex grid system in Qgis

  conda create -n qgis
  conda activate qgis
  conda install qgis
  conda install h3-py

  qgis

From there:
* _Processing tool box >> Open existing script >> Run_, or
* _Plugins >> Python console >> Open editor >> load script >> Run_
